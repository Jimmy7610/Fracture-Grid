# Fracture Grid (Vertical Slice)

Vanilla HTML/CSS/JS + Canvas. Deterministic puzzles via `?seed=`.

## Run
Use any static server. Examples:

### Python
python3 -m http.server 8000

Then open:
http://localhost:8000/?seed=12345

## Controls
- Click adjacent tile: Move
- Click current tile (if Goal): Activate
- S then click cracked tile: Stabilize
- Ctrl+Z: Undo
